Результат должен быть максимально адаптивным. На мобильной верстке:
- уменьшите меню так, чтобы выглядело хорошо
- на 2 экране 3 блока должны ставать один под другим
- большой текст 72 размера должен ставать 36, 48 - 28

По клику на кнопку "Sign Up" в меню должен открываться попап с полями ввода и чекбоксом.
Закрывается попап по клику на крестик.

Для стилей наведения и фокуса используйте UI kit. Крестик должен не только менять цвет, но и складываться в одну линию.
Чекбокс можно не стилизировать.
